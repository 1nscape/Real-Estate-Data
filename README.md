## Description

HORIZON ESTATE  is a service for searching, buying, selling, renting real estate in North America, making mortgages and safely conducting housing transactions.

-  Node.js 
-  Passport.js
-  MySQL
-  Bootstrap
-  Handlebars
-  Realtor.ca API
-  Sequelize
-  Node
-  Dotenv
